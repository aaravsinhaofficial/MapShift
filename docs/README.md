# MapShift Documentation Map

Reviewer-facing artifact instructions live in the repository root:

- `README.md` gives installation, smoke-test, full-study, output, and troubleshooting instructions.
- `ARTIFACT_EVALUATION.md` gives a compact evaluator checklist and expected artifact outputs.
- Versioned configs under `configs/` define the executable benchmark settings used by the release artifact.

The `docs/internal/` directory contains historical planning notes, draft specifications, and release-preparation checklists. These files are retained for provenance only. If an internal note conflicts with the root README, `ARTIFACT_EVALUATION.md`, or versioned configs, the reviewer-facing artifact instructions and configs are authoritative.
